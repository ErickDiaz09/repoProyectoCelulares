# repoProyectoCelulares
En este repositorio se va a realizar el analisis de una base de datos sobre los celulares vendidos en 2025

# Conclusiones 
1. Aunque el precio sea elevado no siginifica que sea bien calificado por las personas, ya sea porque sus capacidades no sean buenas o por distintos factores.
2. Aunque el 5g ya lleve 2 años en operación aún no todos los celulares tienen este soporte.
3. Se puede evidenciar que las marcas lanzan modelos durante todo el año, pero se observa que se lanzan más dispositivos durante el primer y segundo trimestre.
4. La mayoria de los modelos disponibles comercialmente pertenecen a Android,lo que evidencia una oferta mas amplia y diversa en comparación con Apple. 
5. Podemos observar una falta de innovacion en general entre todas las marcas lo que conlleva a que todos los aspectos esten mas o menos a la par y no se presente una amplia dominancia de alguna empresa .
